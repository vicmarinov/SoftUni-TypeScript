TypeScript
==========

This repository contains my solutions to the problems provided during the [TypeScript](https://softuni.bg/trainings/5211/typescript-february-2026) course at [SoftUni](https://softuni.bg).

🚀 How to run the code
--------------------

Each lesson is separated into its own directory and has its own `package.json` and `tsconfig.json` files. The code can be executed using `ts-node`. Alternatively, it can be compiled to JavaScript using `tsc` and then run with `node`.
